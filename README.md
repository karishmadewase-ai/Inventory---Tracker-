# Inventory---Tracker-


A responsive web-based Inventory Management System built using HTML, CSS, and JavaScript. This application helps users manage product inventory efficiently by adding, updating, searching, and deleting products while automatically tracking inventory statistics.

 
  # Features
  
Add new products with name, quantity, and price
Edit existing product details
Delete products from inventory
Real-time search functionality
Automatic calculation of:
Total Products
Total Quantity
Total Inventory Value
User-friendly and responsive interface
Data stored using Local Storage for persistence

# Technologies Used

HTML5
CSS3
JavaScript (ES6)
Local Storage

# Project Objective

The purpose of this project is to provide a simple and effective inventory management solution while demonstrating front-end development skills, DOM manipulation, event handling, and local storage implementation.
